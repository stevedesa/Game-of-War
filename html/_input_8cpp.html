void DealHand ( unsigned int  seed,
Queue< card > &  hand 
)
Author
Steve Nathan de Sa
Description
This functions enters 52 cards into the hand using a random number generator from 0 to 51. the input is a unsigned integer seed provided by the command line arguments. It ses a lookup array to make sure no 2 same numbers are added to the hand.
Parameters
[in]seed- contains seed number for number generator.
[in,out]hand- contains queue of players hand.
Example
Queue<card> hand;
DealHand(123, hand); //Allocates 52 cards to hand.
